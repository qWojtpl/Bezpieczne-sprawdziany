<h1 align="center">
    Bezpieczne sprawdziany
</h1>

### Wymagania

- Android w wersji 10 lub większej (API 29)
- Stały dostęp do Internetu

### Instalacja

1. Pobierz plik <b>pl.securequizapp.apk</b> ze strony <a href="https://github.com/qWojtpl/Bezpieczne-sprawdziany/releases/latest">https://github.com/qWojtpl/Bezpieczne-sprawdziany/releases/latest</a>
2. Otwórz pobrany plik
3. Zezwól na pobieranie aplikacji z tego źródła
4. Postępuj zgodnie z instrukcjami na telefonie
